# JavaScript Default Function Parameters

This JavaScript example demonstrates the use of default parameters introduced in ES6. It contrasts the older ES5 method of manually checking for undefined arguments with the cleaner, more direct ES6 syntax for setting default values in function parameters.

## Language

`javascript`

## How to Run

Save the code as `default_parameters.js` and run it using Node.js: `node default_parameters.js`
